# Test for Kmote with React


## Set up
To set up this project, first clone the repository
```bash
$ git clone 
```

Change your working directory into the project directory
```bash
$ cd 
```
## Run

Install npm modules
```bash
$ npm install
```

Start the application
```bash
$ npm start
```

## Built With
[React](https://github.com/facebook/create-react-app)
